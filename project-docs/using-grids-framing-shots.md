1) Shotcut – how to show the grid

In the main preview window in Shotcut there is a small button under the player labeled “Toggle grid display on the player.”[^1_1][^1_2][^1_3]

- Open your project, look just below the preview monitor for the row of small icons.
- Click the icon that looks like a little grid (this turns the grid overlay on/off).
- Click the tiny arrow next to it to choose the grid type (2×2, 3×3, etc.).[^1_1]

If you do not see that button, make sure you are on a reasonably recent Shotcut version (grid overlays were added in the 18.10 release and are still present in current builds).[^1_3]

2) Olympus E‑PL1 – where to find the grids

The E‑PL1’s composition grids are a Live View overlay you turn on in the menus; they are not a physical screen or separate part.[^1_4][^1_5][^1_6][^1_7]

Two ways to get to them (names may differ slightly depending on language/firmware):

- Via Info settings
    - Press the Menu button.
    - Go to the setup/info section (often called “Info Settings” or similar).
    - Find “Displayed Grid” (you can choose from four grid types or Off).
    - Select the grid style you want (for example a rule-of-thirds style) and confirm.[^1_5][^1_6][^1_4]
    - Exit the menu; the grid will appear over your Live View image.
- Via shooting/live view info cycle
    - While in shooting mode with Live View active, press the INFO button repeatedly.
    - One of the display modes includes the selected framing grid over the image; keep cycling until you see the grid you chose in the menu.[^1_7][^1_4][^1_5]

So on the E‑PL1 the grids “live” in:

- Menu → Info Setting → Displayed Grid (choose one of four).[^1_6][^1_4][^1_5]
- Then toggled in use by pressing INFO in Live View until that overlay shows.[^1_5][^1_7]

- Keep the **text baseline** (bottom of the letters) sitting on Line 2.
- Then keep the subject itself within Lines 1–3 so the whole layout repeats from still to still.


## Quick alignment routine in Shotcut

For each still:

1. Turn on the 3×3 grid.
2. Scale to whatever size you need.
3. Slide the image up/down until:
    - Top of subject ≈ Line 1,
    - Bottom of subject ≈ Line 3.

- Aim for the **top of the body** to sit a little below the **top horizontal line**.
- Aim for the **bottom of the body** to sit a little above the **bottom horizontal line**.
