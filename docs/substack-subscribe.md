# Substack

The [Vintage Reel Service Guides Substack](https://vintagereelserviceguides.substack.com) hosts narrative content about both vintage reels and the Vintage Reel Service Guides project itself. Post series include:

  - **Welcome to the Bench:** This New Arrivals series provides a quick narrative overview of each new reel, some facts about it, history, light on tech-talk.
  - **Service Guides:** This Guides series functions mostly as a notification if you subscribe to the substack, to let you know when a complete set of guides and videos has been published for each reel.
  - **Reel Stories:** This Field Stories series includes reports on field use of the reels after servicing, from test casting sessions in the yard to fishing trips. It also includes more narrative accounts about reels that have some personal stories to tell.
  - **At the Bench:** This Production Notes series talks about the processes behind creating Vintage Reel Service Guides, including photography, video, lighting, software, tools, and other workspace/production space/workstation articles.
