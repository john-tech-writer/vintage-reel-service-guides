# Tools and materials

This page is under development.

## Hand tools

Drivers, wrenches, picks.

[Choosing the right screwdriver](screwdriver-specs.md) is an important basic consideration in any repair project to make the work as easy as possible and avoid damage to the reel.

## Cleaning tools and materials

Containers, brushes, old toothbrushes, picks, rags, mineral spirits, detergents, paper towels.

## Lubrication tools and materials

Basins, greases, oils, rags, paper towels.

## Jigs

Tools you can make for little or no cost. Line winding jig, spring compressing jig (for automatic fly reels).

## Sources

[iFixit](https://www.ifixit.com/Tools/Toolkits) is a good source for tools.
