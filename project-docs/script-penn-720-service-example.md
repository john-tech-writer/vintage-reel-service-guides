Part 3 of 3: Reassembling and testing the reel
Welcome to the Vintage Reel Service Guides. In this video we continue with servicing a Penn 720 spinning reel. This is part 3 of 3, reassembling the reel and testing.
While simply following the disassembly steps in reverse will get the reel put back together, there are a number of notes that will help make sure it actually works correctly when finished. Some of the reassembly steps are sort of mini-tests to check functioning as you go so you won't miss something, then have to disassemble the reel completely when something doesn't work correctly.
1. Smaller end of spool shaft sleeve goes in first.
2. When replacing the dog and spring, make sure the eccentric lever flat side is facing front of housing and the long leaf of the spring presses the back of the housing. Test is using the eccentric lever.
3. Make sure the thrust washer is in place before inserting the main gear.
4. Make sure the pinion gear meshes with the main gear.
5. Make sure the rotor cup washer is seated in place, and note the flat sides of the pinion gear shaft where the rotor cup fits onto it.
6. When fitting crosswind block onto main gear, rotate main gear to position nib at front of housing, fit slot in block over nib.
7. When inserting the spool shaft, push it past the crosswind block first, locate the holes for the connector, and slide it back in place to align the holes. Then insert the connector and work it in.
[Leave the housing plate until last so the gears and rotor cup can be tested and you can see the gears working.]
8. When screwing the handle back onto the housing, make sure to rotate it counter-clockwise.
9. Make sure the housing plate is seated tightly all around--it should snap into place.
10. Rotate spool until it clicks when replacing.
Testing
We can now do some final testing. The main things to check:
1. Make sure the handle turns smoothly in both directions with the anti-reverse off.
2. Make sure the handle will not turn backwards with the anti-reverse on.
3. Make sure you can hear the clicker with the anti-reverse on.
4. Make sure the rotor turns smoothly.
5. Make sure the bail flips correctly.
If all this works, you can call it ready to fish.
Any tight spots or noises at this stage tell you to stop and recheck your work before calling the service complete. Go back through the service procedures as necessary to troubleshoot.
