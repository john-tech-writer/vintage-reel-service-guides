# Subscribe

If you would like to stay updated with more narrative aspects of reel repair—short stories about new reels, behind-the-scenes from the bench, and field reports on using these restored reels to actually fish—you can subscribe to the Vintage Reel Service Guides Substack:

**[→ Subscribe on Substack](https://vintagereelserviceguides.substack.com)**

**What to expect:** There is no cost at this time. You will not be flooded with posts. These will be done on an occasional, fairly random schedule, as time permits.

*Note: You can browse posts without subscribing, but you'll need to create a free Substack account to view them.*
